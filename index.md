# HomeCore

HomeCore は、在庫管理・TODO 管理・買い物管理・掃除管理をひとつにまとめた iOS アプリです。iCloud を利用したデータ同期・共有と、選択した機能画面を自動で切り替えて表示するスライドショー機能を提供します。

HomeCore is an iOS app that brings Stock, TODO, Shopping, and Cleaning management together in one place. It supports iCloud-based sync and sharing, plus a slideshow feature that automatically cycles through selected feature screens.

## 主な機能 / Key Features

- 在庫管理・TODO 管理・買い物管理・掃除管理を 1 つのアプリで管理できます。
- iCloud 共有を利用して、家族や共有参加者とデータを自動同期できます。
- プレミアム版では、選択した機能画面を全画面で自動切り替え表示するスライドショー機能を利用できます。
- 毎日のローカル通知で、HomeCore Daily Summary を受け取れます。
- プレミアム版は買い切り型で、購入の復元に対応しています。
